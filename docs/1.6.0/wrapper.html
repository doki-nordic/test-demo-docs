MCUboot documentation¶

This documentation is based on the official MCUboot documentation, but focuses on the use of MCUboot in nRF Connect SDK.

The documentation includes documentation files from the upstream MCUboot repository. Topics that are not relevant for the integration of MCUboot in the nRF Connect SDK are excluded from the navigation.

Using MCUboot in nRF Connect SDK contains information that is relevant for nRF Connect SDK’s fork of MCUboot but not for the upstream MCUboot repository.
